# Sistema de Gestión de Cuotas de Mantenimiento - Condominio Vista Verde

## descripción del condominio

El Condominio Vista Verde es un conjunto residencial ubicado en \[Guatemala/Ciudad de Guatemala], conformado por 30 casas numeradas del 1 al 30. Cada casa paga una cuota de mantenimiento mensual para cubrir los gastos comunes del condominio, como seguridad, limpieza, áreas verdes, mantenimiento de infraestructura y servicios generales.

### **Datos del Condominio**

Nombre

Vista Verde

Número de casas

30 (numeradas del 1 al 30)

Cuota inicial

Q1,500.00 mensuales

Recaudación mensual esperada

Q45,000.00

### **Objetivo del Sistema**

**El sistema permite al administrador del condominio gestionar de manera eficiente:**

* Registro de propietarios por casa
* Control de pagos de cuotas de mantenimiento
* Configuración del monto de la cuota
* Consulta de estado de cuenta por casa
* Generación de reportes generales
* Identificación de casas morosas
* Eliminación de propietarios (solo si están solventes)

## **Beneficios del sistema**

## **Instrucciones para Ejecutar el Proyecto**

### **Requisitos Previos**

##### **(1) \*\*Tener Java JDK 21 o superior instalado.\*\***

* **Windows**

https://www.oracle.com/java/technologies/downloads/#jdk26-windows

* **MAC**

https://www.oracle.com/java/technologies/downloads/#jdk26-mac

* **Linux**

https://www.oracle.com/java/technologies/downloads/#jdk26-linux

**Para verificar, abre una terminal y escribe:**
java -versión

##### **(2) Descomprimir**

Descomprime el archivo descargado

##### **(3) Estar dentro de la carpeta Ejecutable**

Proyecto\_final\_Programacion-main\\Proyecto\_final\_Programacion-main\\ejecutable

##### **(3) Ejecutar el programa**

###### **WINDOWS**

* **Opción 1: Terminal**

(1) Abre la carpeta donde está el .jar (ejecutable/)

(2) Escribe cmd en la barra de direcciones y presiona Enter

(3) Ejecuta:

java -jar SistemaDeAdministracion-1.0-SNAPSHOT-jar-with-dependencies.jar

* **Opción 2: Configurar "Abrir con" (solo una vez**)

(1) Haz clic derecho en el .jar → "Abrir con" → "Elegir otra aplicación"

(2) Selecciona "Java(TM) Platform SE binary"

(3) Marca "Usar siempre esta aplicación"

(4) Aceptar → luego solo doble clic

**Opción 3: Script ejecutable**

(1) Haz doble clic en VistaVerde-Windows.bat

##### **MAC**

* **opción 1: Terminal**

(1) Abre la carpeta donde está el .jar (ejecutable/)

(2) Haz clic derecho → "Nueva terminal en carpeta"

(3) Ejecuta:

java -jar SistemaDeAdministracion-1.0-SNAPSHOT-jar-with-dependencies.jar

* **opción 2 Abrir con**

(1) Haz clic derecho en el .jar → "Abrir con" → "Java"

* **Opción 3: Script ejecutable**

(1) Abre la Terminal en la carpeta ejecutable/

(2) Da permisos (solo la primera vez):

chmod +x VistaVerde-Mac.command

(3)Haz doble clic en VistaVerde-Mac.command

&#x09;Si no se abre: Haz clic derecho → "Abrir" → "Abrir"

##### **LINUX**

**Opción 1: Terminal**

(1) Abre la carpeta donde está el .jar (ejecutable/)

(2) Haz clic derecho → "Abrir en terminal"

(3) Ejecuta:

java -jar SistemaDeAdministracion-1.0-SNAPSHOT-jar-with-dependencies.jar

**Opción 2: Script ejecutable**

(1)Abre la Terminal en la carpeta ejecutable/

(2) Da permisos (solo la primera vez):

(3)Haz doble clic en VistaVerde-Linux.sh → "Ejecutar en la terminal"
